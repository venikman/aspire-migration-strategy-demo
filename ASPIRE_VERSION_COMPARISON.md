**Aspire 9.5**:
- Solid foundation
- Manual configuration required
- Good for .NET 9 projects
- Production-ready but verbose

**Aspire 13** (Our Project):
- Modern features (Vite, resilience, sampling)
- Less boilerplate code
- Better performance (40% dashboard, 2x startup)
- Cost-effective (90% savings with smart sampling)
- Production-ready out of the box

**Migration**: Straightforward for most projects
- Update .NET SDK to 10+

- Replace AddNpmApp with AddViteApp
- Add ASPIRE_ALLOW_UNSECURED_TRANSPORT flag
- Enjoy automatic resilience and better performance!
